# Regras de Uso do Logo

- **Área de proteção:** mínimo a altura do "W" ao redor
- **Tamanho mínimo:** 80px largura (horizontal) · 60px altura (vertical)
- **Nunca:** distorcer, girar em diagonal, adicionar sombras, mudar cores fora das versões oficiais
- **Sempre:** usar PNG com fundo transparente
