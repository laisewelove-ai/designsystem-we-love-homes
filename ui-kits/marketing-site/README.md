# Marketing site UI kit

A high-fidelity recreation of the We Love Homes marketing site homepage. Demonstrates the brand's core marketing surfaces:

- `Header.jsx` — sticky translucent top nav
- `Hero.jsx` — full-bleed hero with mural background, brushy "em Santiago" display headline, and inline 4-field search bar
- `ListingCard.jsx` — apartment card with photo, rating, amenities, price
- `Sections.jsx` — featured listings, how-it-works, neighborhoods grid, coral CTA band, navy footer
- `styles.css` — extends `colors_and_type.css`

**No real codebase or Figma was provided** — these screens are inferred from the brand kit. Treat as a strong starting point, not pixel-fidelity.

Open `index.html` to view.
