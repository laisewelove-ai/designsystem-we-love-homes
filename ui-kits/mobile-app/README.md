# Mobile app UI kit

A clickable recreation of the We Love Homes mobile app. Tap a listing to see details, then "Reservar" to see confirmation, then "Ver minha viagem" for the trip itinerary.

- `AppScreens.jsx` — Home (search/discover), Listing detail, Confirmed, Trip
- `styles.css` — extends `colors_and_type.css`
- `index.html` — runs the app in an iPhone-style frame

**No real codebase or Figma was provided** — these screens are inferred from the brand kit. Treat as a strong starting point, not pixel-fidelity.
